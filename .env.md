## Visao geral
O projeto `renovajf-site` e uma landing page em **React + Vite** com **Tailwind CSS** e componentes reutilizaveis em `src/components`.

### Stack atual
- React 18
- Vite 5
- Tailwind CSS 3
- Swiper
- react-icons

### Scripts disponiveis
- `npm run dev`
- `npm run build`
- `npm run preview`

## Status de ambiente
O projeto **ja consome variaveis de ambiente** via `import.meta.env`.

As configuracoes centralizadas ficam em:
- `src/config/site.js`

Arquivo de referencia para novos ambientes:
- `.env.example`

Variaveis usadas atualmente:
- `VITE_SITE_NAME`
- `VITE_WHATSAPP_NUMBER`
- `VITE_WHATSAPP_LINK`
- `VITE_CONTACT_PHONE`
- `VITE_CONTACT_EMAIL`
- `VITE_CONTACT_ADDRESS`
- `VITE_INSTAGRAM_URL`
- `VITE_FACEBOOK_URL`
- `VITE_CATALOG_PDF`

## Mapa funcional por arquivo

### Entrada da aplicacao
- `src/main.jsx`
  - Renderiza a aplicacao React no elemento `#root`.
- `src/App.jsx`
  - Compoe a pagina principal.
  - Importa as secoes da home.
  - Usa o link do WhatsApp vindo de `siteConfig`.

### Configuracao de ambiente
- `src/config/site.js`
  - Centraliza o acesso a `import.meta.env`.
  - Aplica fallbacks para manter o site funcionando sem `.env` local.
- `.env.example`
  - Modelo para criar um `.env` novo.

### Componentes de interface
- `src/components/Header.jsx`
  - Header fixo com menu desktop e mobile.
  - Usa `useState` e `useEffect` para controlar menu e efeito de scroll.
- `src/components/Hero.jsx`
  - Hero com troca automatica de imagem.
  - CTA para WhatsApp via `siteConfig`.
- `src/components/About.jsx`
  - Secao institucional com historia, missao e visao.
- `src/components/Gallery.jsx`
  - Galeria com preview, modal e botao ver mais/ver menos.
  - Faz fallback de `.jpg` para `.png`.
- `src/components/Services.jsx`
  - Lista de servicos.
  - Usa `Swiper` para parceiros.
  - Link para o catalogo em PDF via `siteConfig`.
- `src/components/Contact.jsx`
  - Dados de contato e links sociais via `siteConfig`.
  - Formulario visual, sem envio implementado.
- `src/components/Footer.jsx`
  - Rodape com ano dinamico.

### Estilos globais
- `src/index.css`
  - Tailwind base/components/utilities.
  - Ajustes globais de fundo, cores e animacoes.
- `src/App.css`
  - Arquivo presente, mas nao foi identificado como peГ§a central da estrutura atual.

### Configuracoes
- `vite.config.js`
  - Configuracao minima do Vite com plugin React.
- `tailwind.config.js`
  - Cores e fontes personalizadas.
- `postcss.config.js`
  - Tailwind + Autoprefixer.

## Exemplo de uso no codigo
O projeto resolve assim:

```js
const whatsappLink = env.VITE_WHATSAPP_LINK ?? `https://wa.me/${whatsappNumber}`;
```

Isso permite sobrescrever a URL direto no ambiente, mas manter um fallback seguro para desenvolvimento.
